# 💸 Controle de Gastos Pessoais

Um aplicativo web clean e intuitivo para gerenciamento de finanças pessoais. Desenvolvido com o auxílio da inteligência artificial do **Lovable**, este projeto permite que você registre despesas, defina seu salário e visualize exatamente quanto está economizando a cada mês.

🌐 **Acesse o projeto online:** [Controle de Gastos - Lovable App](https://controledemy-cents-keeper-19.lovable.app/)

## ✨ Funcionalidades

* 📊 **Painel de Resumo (Dashboard):** Acompanhe em tempo real:
    * Seu salário do mês atual.
    * O total gasto por você.
    * Valores que terceiros devem a você (divisão de contas).
    * Sobra estimada com base nos seus ganhos e gastos.
* ➕ **Cadastro de Despesas Rápido:** Formulário prático para adicionar despesas contendo descrição, valor, pagador (Você ou Terceiros) e categoria (ex: Cartão de Crédito, Mercado, Faculdade).
* 📅 **Navegação Mensal:** Filtre e visualize suas finanças por mês e ano específicos.
* 🔒 **Privacidade e Armazenamento Local:** Sem necessidade de login ou banco de dados externo. Seus dados são salvos automaticamente no *Local Storage* do seu próprio navegador (Totalmente privado).

## 🛠️ Tecnologias Utilizadas

* **[Lovable](https://lovable.dev/):** Plataforma/IA utilizada para o desenvolvimento e geração da interface UI/UX.
* **Web Standard:** HTML, CSS e JavaScript (React / Vite / Tailwind CSS).
* **Local Storage API:** Para persistência de dados no navegador do usuário.

## 🚀 Como Usar

1.  Acesse a aplicação clicando [neste link](https://controledemy-cents-keeper-19.lovable.app/).
2.  No painel inicial, clique na área de **Salário do mês** para definir o seu rendimento (ou ajuste conforme necessário).
3.  Utilize o formulário **"Nova despesa"** para registrar seus gastos diários.
4.  Selecione se a despesa foi paga por você ou por outra pessoa (ideal para controle de rachadinhas/dívidas de amigos).
5.  Acompanhe a lista em **"Despesas do mês"** e o cálculo automático da sua sobra estimada.

---
*⚠️ Nota de Privacidade: Todos os seus dados ficam salvos apenas no seu navegador local. Se você limpar o cache ou os dados de navegação do site, suas informações registradas serão perdidas. Nenhuma informação é enviada para servidores externos.*
